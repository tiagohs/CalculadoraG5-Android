# CalculadoraG5-Android


Uma simples implementação de uma calculadora cêntifica utilizando várias Libraries e APIs de Android.

###Padrões de Projetos Adotados:

<ul>
<li><a href="http://antonioleiva.com/mvp-android">MVP</a> - Model View Presenter.</li>
</ul>

###Libaries utilizadas nesse Projeto:

<ul>
<li><a href="https://github.com/square/dagger">Dagger</a> - para Injeção de depêndencias.</li>
<li><a href="https://github.com/JakeWharton/butterknife">Butterknife</a> - para binding minhas Views</li>
</ul>
